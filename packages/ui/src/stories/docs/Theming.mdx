# Theming

This library provides flexible theming through `Theme` context props and semantic CSS variable control.

## Theme Component

The `Theme` component provides context for all child components:

```tsx
import { Theme } from "@incmix/ui";

<Theme
  appearance="light"
  accentColor="blue"
  radius="md"
  scaling="100%"
>
  {children}
</Theme>
```

### Nested Themes

You can nest Theme components to create sections with different styles:

```tsx
<Theme accentColor="blue">
  <Button>Blue Button</Button>

  <Theme accentColor="green">
    <Button>Green Button</Button>
  </Theme>
</Theme>
```

## Theme Context

Access the current theme values with `useThemeContext`:

```tsx
import { useThemeContext } from "@incmix/ui";

function ThemeDisplay() {
  const theme = useThemeContext();

  return (
    <div>
      <p>Mode: {theme.appearance}</p>
      <p>Accent: {theme.accentColor}</p>
      <p>Radius: {theme.radius}</p>
    </div>
  );
}
```

## CSS Variables

The theme system uses semantic CSS custom properties. Override the semantic lanes rather than legacy HSL aliases:

```css
:root {
  --color-primary-solid: oklch(0.61 0.11 222);
  --color-primary-text: oklch(0.44 0.11 222);
  --color-primary-contrast: oklch(0.98 0.02 201);
  --color-neutral-surface: oklch(1 0 0);
  --color-neutral-text: oklch(0.22 0 0);
  --color-neutral-border: color-mix(in oklch, black 12%, white);
  --radius: 0.5rem;
}

.dark {
  --color-neutral-surface: oklch(0.23 0 0);
  --color-neutral-text: oklch(0.95 0 0);
  --color-neutral-border: color-mix(in oklch, white 20%, black);
}
```

### Color Format

Colors use CSS color values directly:

```css
--color-primary-solid: oklch(0.62 0.18 250);
```

In Tailwind classes, use:

```tsx
<div className="bg-primary text-primary-contrast" />
```

## Dark Mode

### Automatic Detection

Use `appearance="inherit"` to follow system preference:

```tsx
<Theme appearance="inherit">
  {children}
</Theme>
```

### Manual Toggle

```tsx
function DarkModeToggle() {
  const [dark, setDark] = React.useState(false);

  return (
    <Theme appearance={dark ? "dark" : "light"}>
      <Button onClick={() => setDark(!dark)}>
        Toggle Dark Mode
      </Button>
      {children}
    </Theme>
  );
}
```

### CSS Class

The Theme component adds `.dark` class when in dark mode, compatible with Tailwind's dark mode:

```js
// tailwind.config.js
module.exports = {
  darkMode: "class",
  // ...
}
```
