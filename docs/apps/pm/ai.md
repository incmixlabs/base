
⏺ Strategy & Prioritization
  - Auto-prioritize backlog using impact/effort scoring from historical velocity data
  - Predict feature adoption rates from usage patterns of similar features
  - Opportunity scoring — rank feature requests by revenue potential, churn risk, market signals
  - Competitive gap analysis — extract feature sets from competitor docs/changelogs

  User Research & Insights
  - Cluster user feedback (support tickets, reviews, surveys) into themes
  - Sentiment analysis on feature requests — detect urgency/frustration
  - Auto-summarize user interviews into actionable insights
  - Identify unmet needs from behavioral data (where users drop off, rage-click)

  Requirements & Specs
  - Generate PRDs from rough bullet points or voice notes
  - Extract acceptance criteria from user stories
  - Detect conflicting requirements across stakeholders
  - Auto-link requirements to existing features/components

  Roadmap & Planning
  - Scenario modeling — "if we ship X first, what happens to Y timeline?"
  - Dependency detection across features and teams
  - Resource allocation optimization given constraints
  - Release risk scoring based on scope, team capacity, technical debt

  Go-to-Market
  - Auto-generate release notes from commit history + PRDs
  - Predict best launch timing from market/seasonal data
  - Generate positioning/messaging variants for different personas
  - Competitive positioning maps from feature comparisons

  Metrics & Analytics
  - Anomaly detection on product metrics (sudden drop in activation, usage cliff)
  - Attribution modeling — which features drive retention/revenue
  - Churn prediction tied to feature usage gaps
  - Auto-generate executive summaries from dashboards
