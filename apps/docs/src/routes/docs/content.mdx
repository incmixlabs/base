  <Text size="md" className="max-w-3xl text-muted-foreground">

<section className="mt-10 space-y-4">
  <Heading as="h2" size="lg">
    What It Tries To Do
  </Heading>
  <div className="space-y-3">
    <Text size="md">
      <strong>Make customization structural.</strong> The system is token-based by default, so teams can shape color,
      spacing, type, radius, and density without forking the component layer.
    </Text>
    <Text size="md">
      <strong>Make responsiveness belong to components.</strong> Container queries are used where they matter, so cards,
      containers, and forms respond to the space they are actually given.
    </Text>
    <Text size="md">
      <strong>Make rich interfaces easier to declare.</strong> A composed schema approach supports forms and structured
      content, not just isolated UI primitives.
    </Text>
  </div>
</section>
