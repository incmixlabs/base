<Heading as="h1" size="2x">
  Runtime
</Heading>
<Text size="sm" className="mt-2 text-muted-foreground">
  AutoForm now uses an explicit split: the shared runtime owns submit lifecycle, validation status, touched/dirty metadata,
  and condition evaluation; the local form runtime owns value and error bags; the renderer owns active-branch filtering and
  submit-time pruning.
</Text>

<Heading as="h2" size="lg" id="ownership-boundary" data-heading>
  Ownership Boundary
</Heading>

<Text size="sm" className="mt-2 text-muted-foreground">
  This contract is intentional. `values`, validation errors, server errors, and form-level errors stay local so field updates,
  controlled values, and submit helpers remain synchronous and predictable. Shared runtime state covers lifecycle semantics:
  `isSubmitting`, validation phases/results, touched/dirty tracking, and condition-driven visibility. `AutoFormModelRenderer`
  then filters inactive-branch errors from presentation and prunes inactive branch values before submit.
</Text>

<Heading as="h2" size="lg" id="async-submit" data-heading>
  Async Submit Lifecycle
</Heading>

<LiveCodeBlock
  scope={{ normalizeJsonSchemaWithUiSchema, AutoFormModelRenderer, autoformRendererJsonSchema, autoformRendererUiSchema }}
  initialCode={`export default function Example() {
  const [submitted, setSubmitted] = React.useState(null)
  const model = normalizeJsonSchemaWithUiSchema(
    autoformRendererJsonSchema,
    autoformRendererUiSchema,
  )

  return (
    <div className="space-y-4">
      <AutoFormModelRenderer
        model={model}
        surface="inline"
        validationSchema={autoformRendererJsonSchema}
        submitLabel="Create trip"
        defaultValues={{
          traveler: {
            fullName: "Ada Lovelace",
            email: "ada@example.com",
          },
          country: "US",
          startAt: "2026-03-20T10:00:00.000Z",
        }}
        onSubmit={async values => {
          await new Promise(resolve => window.setTimeout(resolve, 1200))
          setSubmitted(values)
        }}
      />
      <pre className="overflow-auto rounded-xl bg-muted/60 p-3 text-xs leading-5 text-foreground">
        {JSON.stringify(submitted, null, 2)}
      </pre>
    </div>
  )
}`}
/>
