# Media Player

A compound video and audio player with transport controls, volume, captions, playback speed, fullscreen, picture-in-picture, and settings surfaces.

## Keyboard Shortcuts

| Key | Action |
| --- | --- |
| `Space` / `K` | Play / Pause |
| `←` / `→` | Seek backward / forward 5s |
| `↑` / `↓` | Volume up / down |
| `J` / `L` | Seek backward / forward 10s |
| `M` | Mute / Unmute |
| `F` | Toggle fullscreen |
| `P` | Toggle Picture-in-Picture |
| `C` | Toggle captions |
| `R` | Toggle loop |
| `D` | Download |
| `<` / `>` | Decrease / Increase playback speed |
| `0-9` | Seek to 0%-90% |
| `Home` / `End` | Seek to start / end |

## Auto Hide

Set `autoHide` on the root to hide controls after inactivity during playback.

```tsx
<MediaPlayer.Root autoHide>
  {/* ... */}
</MediaPlayer.Root>
```
